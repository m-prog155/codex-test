# 论文格式严审记录（2026-04-16）

## 1. 审查依据

截至 2026-04-16，本轮格式审查主要依据以下标准信息进行：

1. 学位论文结构与编排：GB/T 7713.1-2025《信息与文献 编写规则 第1部分：学位论文》
2. 参考文献著录：GB/T 7714-2015《信息与文献 参考文献著录规则》
3. 标点符号使用：GB/T 15834-2011《标点符号用法》

说明：

- 2026-04-16 这一天，GB/T 7714-2015 仍为现行标准。
- GB/T 7714-2025 已发布，但实施日期为 2026-07-01，因此当前论文参考文献仍应按 2015 版执行。
- 学校模板在封面、题名页、声明页、页眉页脚、字号行距等具体落版细节上通常优先于通用国标示例，因此最终提交版仍需以学校 Word 模板做最后确认。

### 2.1 文献编号顺序问题已修正

此前总稿中 CCPD 原始论文虽然在第 1 章较早出现，但被放在参考文献末尾并编号为 [21]，不符合顺序编码制的常见要求。

本轮已完成以下修正：

- 将 CCPD 原始论文调整为 [4]
- 将后续参考文献编号整体顺延
- 同步修正正文中的对应引用号

当前总稿中的文献编号已按正文首次出现顺序重排。

### 2.2 Markdown 草稿痕迹已清理一轮

本轮已在总稿和摘要草案中清理以下问题：

- 删除总稿顶部“论文正文总稿（第1章至第5章）”草稿标题
- 去除正文与摘要中的反引号包裹写法
- 将“smoke 验证”等明显口语化、工程草稿化表述改为“联调验证”等正式写法

### 2.3 封面稿定位已澄清

封面稿当前仅作为封面文案参考，不等同于正式题名页。该说明已补入封面草案，避免后续把“封面”误当成“题名页”直接提交。

## 3. 目前仍未解决的格式风险

以下问题不是内容错误，而是最终交稿前必须继续处理的格式风险。

### 3.1 当前仍不是 Word 定稿版

目前项目中的论文文件仍以 Markdown 草稿为主，尚未进入学校 Word 模板。因此以下项目当前无法视为“已通过格式审查”：

- 页边距
- 页眉页脚
- 页码位置与分页方式
- 正文字号、行距、首行缩进
- 各级标题样式
- 表格三线表样式
- 参考文献悬挂缩进
- 目录自动生成效果

结论：

当前只能完成“内容结构与文字格式预审”，不能替代“Word 落版后的终审”。

### 3.2 题名页尚未形成正式版本

按现行学位论文编写规则，题名页属于正式构成要素；封面可视学校要求处理，但题名页不能缺失。

当前仓库中已有：

- 封面草案
- 中文摘要草案
- 英文摘要草案
- 第 1 章至第 5 章正文草案

当前仍缺：

- 与学校模板一致的正式题名页
- 可能需要的原创性声明页
- 可能需要的使用授权说明页

这些内容是否必须单独成页，最终以学校模板和指导教师要求为准，但从国标视角看，不能只保留一个“封面文案”就结束。

### 3.3 图表落版仍未完成

目前正文已包含表 4-1、表 4-2、表 4-3 的文字与表格内容，但图片/视频样例分析部分仍以文字描述为主。

如果最终论文准备插入系统运行截图、图片样例标注图或视频帧截图，则需要补齐：

- 图号，例如“图 4-1”“图 4-2”
- 图题
- 正文中的图文对应引用
- 与学校模板一致的图注位置和字体格式

### 3.4 参考文献仍需做 Word 落版检查

当前总稿中的参考文献内容已基本齐全，编号顺序也已修正，但在最终交稿前仍需在 Word 中检查以下事项：

- 是否采用学校要求的编号样式
- 是否使用悬挂缩进
- 中英文标点与空格是否统一
- 电子文献是否需要保留 DOI 或访问路径
- 文献列表是否与正文最终引用完全一致

## 4. 当前审查结论

从“内容结构 + 基本文字格式”角度看，当前论文草稿已经明显比前一版更规范，尤其是：

- 文献编号顺序问题已修正
- CCPD 原始文献出处已补齐
- 明显的 Markdown 痕迹已清理一轮
- “封面”和“题名页”概念已明确区分

但从“最终提交格式”角度看，当前仍不能判定为完全符合国家标准和学校格式要求，原因很明确：

- 还没有进入正式 Word 模板
- 题名页、声明页、目录页等前置部分尚未整体组装
- 图表、页码、段落样式、参考文献版式仍未完成最终落版核查

## 5. 下一步建议

建议按以下顺序继续收口：

1. 将封面、题名页、摘要、目录、正文、参考文献组装进学校 Word 模板
2. 在 Word 中统一标题样式、页码、段落格式和图表格式
3. 落版完成后再做一轮终审，只查格式，不再改内容

只有完成这一步，才能真正判断“格式是否符合最终提交要求”。
