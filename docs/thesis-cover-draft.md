# 论文封面草案（参考学长样式）

说明：
- 本稿参考学长学姐正文封面的主流样式整理，字段顺序与排版口径尽量贴近学校常见本科论文封面。
- 按现行学位论文编写规则，封面与题名页不是同一概念；当前文件仅提供封面文案，不替代正式题名页。
- 当前已直接采用 `阳 光 学 院` 和 `本科毕业论文(设计)` 的写法。
- 当前封面核心字段已基本补齐。
- 正式落到 `Word` 时，建议按学校模板处理字号、居中、行距和下划线。

可直接复制版：

```text
阳 光 学 院

本科毕业论文(设计)

题    目：基于深度学习的车辆类型与车牌检测识别系统

院(系)别：人工智能学院

专    业：数据科学与大数据技术专升本

年    级：2024级

学    号：2024349243

姓    名：陈东东

指导教师：陈志聪教授    陈辉煌老师

2026年5月
```

推荐填写说明：
- `指导教师`：当前按你提供的信息写为 `陈志聪教授    陈辉煌老师`
- `日期`：当前按你给的信息写为 `2026年5月`

如果你们学院最后给的是另一版封面模板，通常只会改这几个点：
- `院(系)别` 改成 `学    院`
- `指导教师` 后面是否保留括号职称
- 日期写成 `2026 年 5 月` 或 `2026年    月    日`
